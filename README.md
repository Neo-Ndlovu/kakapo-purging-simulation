# Kākāpō Purging Simulation

## Description
A Wright-Fisher forward simulation modelling the purging of fully recessive lethal mutations during a population bottleneck, parameterized to reflect the demographic history of the Kākāpō (*Strigops habroptilus*).

Neo Ndlovu and Megan Ebbeling
